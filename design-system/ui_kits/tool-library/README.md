# Kipfoot Tool Library UI Kit

## Overview
The tool library / browse page. Neal.fun-style visual cards grouped by discipline. Filter sidebar on the left, 3-col card grid on the right.

## Screens
1. **Library Header** — Title, search input, view toggle
2. **Filter Sidebar** — Discipline checkboxes, status filters
3. **Tool Grid** — 3-col grid of ToolCards
4. **Tool Card** — Thumbnail, name, description, discipline tag, version

## Components
- `LibraryHeader` — search, filters, count
- `FilterSidebar` — discipline + status filters
- `ToolGrid` — responsive card grid
- `ToolCard` — full card component
- `EmptyState` — no results
