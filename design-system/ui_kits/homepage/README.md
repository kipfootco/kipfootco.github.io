# Kipfoot Homepage UI Kit

## Overview
The homepage is the entry point and app launcher. Raycast-inspired: restrained shell, dark hero with a warm radial glow, centered content, and a colorful tool carousel below the fold.

## Screens
1. **Hero** — Wordmark, headline, short descriptor, primary CTA, secondary CTA
2. **Tool Carousel** — Featured tools in colorful accent cards
3. **Tool Grid Section** — Discipline-grouped tool cards (neal.fun style)
4. **Footer** — Minimal: logo, links, version

## Components
- `TopNav` — transparent → frosted on scroll
- `HeroSection` — centered, large display type, warm glow
- `ToolCarousel` — horizontal scroll of accent cards
- `DisciplineSection` — label + 3-col tool card grid
- `Footer` — minimal dark footer
