Why deflection controls this design Learn

For longer spans with lighter loads, serviceability (deflection) often governs over strength. At L = 24 ft, the L/360 limit for live load deflection is 0.80 in. The computed δ = 0.89 in exceeds this by 11%.

Options: increase Ix by upsizing to W21×68 (Ix = 1,600 in⁴), reduce the live load, or increase the span depth ratio. The W18×97 provides adequate strength but insufficient stiffness at this span.

Flexure calculation trace Equation
wu = 1.2D + 1.6L = 1.2(1.5) + 1.6(2.0) = 5.60 k/ft
Mu = wu·L² / 8
Mu = 5.60 × 24² / 8 = 403.2 kip·ft (factored)
ϕMn = 0.90 × Fy × Zx = 0.90 × 50 × 211 / 12 = 791.3 kip·ft
D/C = 403.2 / 791.3 = 0.51
